# VariableSetting 開発計画（初期版）

## プロジェクト概要

**プラグイン名**: VariableSetting  
**目的**: Tailwind CSS/Shadcn UI に準拠したデザインシステムを数クリックで生成  
**主な機能**: プライマリカラーからのデザインシステム生成、変数コレクション構築、コンポーネント生成、開発者エクスポート  
**アプローチ**: インサイドアウト（基本要素から体系的に構築）  
**対象ユーザー**: UI/UXデザイナー、特に Tailwind/Shadcn を使用する開発チームと協業するデザイナー

## 1. 実装計画

### 1.1 フェーズ1: 基盤構築

**目標**: 開発環境の整備とプロジェクト基盤の確立

- **タスク**:
  - **基盤構築**:
    - TypeScript + Vite の開発環境セットアップ
    - ESLint + Prettier の設定
    - 基本的なCI/CD（GitHub Actions）設定

  - **最小限のUI構築**:
    - シンプルなHTML/CSS/JSベースのUI設計
    - カラーピッカーとHEXコード入力機能実装

  - **コアロジック実装**:
    - カラー演算ユーティリティの開発
    - Figma Plugin API連携の基本実装

- **成果物**:
  - 動作する基本プラグイン
  - 基本的なコード構造

### 1.2 フェーズ2: カラーシステム実装

**目標**: 基本的なカラーパレット生成機能の実装

- **タスク**:
  - **カラーシステム**:
    - プライマリカラーからの10段階カラーパレット生成ロジック
    - カラースウォッチ生成と視覚的表示
    - 基本色調整機能

  - **技術アーキテクチャ整備**:
    - フロントエンド：バニラHTML/CSS/JavaScript
    - バックエンド：TypeScript、Figma Plugin API
    - シンプルなメッセージパッシング設計

- **成果物**:
  - カラーパレット生成機能
  - カラースウォッチビューア
